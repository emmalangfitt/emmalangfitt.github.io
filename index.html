    development

    Vibes Mobile App

    • I worked in a group of five to develop a mobile app for creating location-based collaborative playlists, built with React Native using Spotify's Web API
    • The app allows you to be authenticated with Spotify and play music through Spotify's player
    • Users can create a playlist and select a starting genre/vibe, which will populate the initial playlist. This playlist will be location-based, in that users in the vicinity of a playlist will automatically be able to add to it, and once they leave, they no longer have access.

    Nosedive Mobile Research App

    • I built a social rating app based on the Black Mirror episode Nosedive for use in Dartmouth's Computational Social Affective Neuroscience Laboratory's social experiments
    • Users can create an account, view the ratings of the other participants in their experimental group, search for participants by name or participant number, and rate other participants on a five-point scale
    • Admin users can create experiment groups and change the experimental conditions by using a weighted vs. non-weighted rating systems and starting participants with equal or varied ratings
    • Built with the Ionic development framework to create a hybrid mobile app using HTML, CSS, and Javascript

    Mindful Social Media Chrome Extension

    • I created a Google Chrome Extension using JavaScript and CSS to automatically turn social media sites to greyscale, encouraging mindful use of these platforms by removing color cues that create compulsive attention
    • This was inspired by the work of Tristan Harris, who promotes conscious use of technology and ethical deisgn
